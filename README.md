# Blog

Here's my personal blog. I'll talk about life, and tech. You can find my weekly coding blogs on my [dev community page]

This blog was built with [Abelljs Blog Template]

# :computer: Building locally

1. clone this repo

`git clone url`

2. go into the cloned folder

`cd blog`

3. Install the needed NPM pacakges

`npm i`

4. Start your local development server

`npm run dev`

5. Open [localhost]

# :rocket: Bulding for production

1. clone this repo

`git clone url`

2. go into the cloned folder

`cd blog`

3. Build the files

`npm run build`

4. Go into the build folder.

`cd dist`
